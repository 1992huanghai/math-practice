# 小学生加减法练习 App

一个专为 iPad 设计的加减法计算练习应用，适合小学生使用。

## 功能特点

- ✅ **多种难度选择**
  - 20以内加减法（适合一年级）
  - 100以内加减法（适合二年级）
  - 1000以内加减法（适合三年级）

- ✅ **智能题目生成**
  - 随机生成加法和减法题目
  - 确保答案在难度范围内
  - 减法题目保证结果为正数

- ✅ **实时反馈**
  - 答对显示绿色提示和动画效果
  - 答错显示正确答案和提示
  - 即时统计正确/错误/总数

- ✅ **iPad 优化设计**
  - 大按钮，易于点击
  - 清晰的字体和布局
  - 响应式设计，支持横屏和竖屏
  - 防止页面缩放，提供原生应用体验

## 使用方法

### 在 iPad 上使用

1. **方法一：通过 Safari 浏览器**
   - 在 iPad 上打开 Safari 浏览器
   - 访问应用页面（需要部署到服务器或使用本地服务器）
   - 点击 Safari 的分享按钮
   - 选择"添加到主屏幕"
   - 即可像原生应用一样使用

2. **方法二：本地开发服务器**
   ```bash
   # 使用 Python 3
   cd math_practice
   python3 -m http.server 8000
   
   # 或使用 Node.js
   npx http-server -p 8000
   ```
   - 在 iPad Safari 中访问：`http://你的电脑IP:8000`

### 操作说明

1. **选择难度**：点击对应的难度按钮开始练习
2. **输入答案**：在输入框中输入答案（支持键盘输入）
3. **提交答案**：点击"提交答案"按钮或按回车键
4. **下一题**：答完后点击"下一题"继续练习
5. **返回**：随时可以返回选择其他难度

## 部署到 Web Server（获得域名）

这个应用可以免费托管到多个平台并获得域名。详细部署指南请查看 [DEPLOY.md](./DEPLOY.md)

### 快速部署（推荐）

**Netlify（最简单）：**
1. 访问 https://www.netlify.com
2. 注册/登录账号
3. 点击 "Add new site" → "Deploy manually"
4. 将整个 `math_practice` 文件夹拖进去
5. 完成！获得免费域名（如：`你的项目名.netlify.app`）

**其他平台：**
- **GitHub Pages** - 免费域名：`你的用户名.github.io/math-practice`
- **Vercel** - 免费域名：`你的项目名.vercel.app`
- **Cloudflare Pages** - 免费域名：`你的项目名.pages.dev`

所有平台都支持：
- ✅ 免费 HTTPS
- ✅ 绑定自定义域名
- ✅ 自动部署（连接 Git 后）
